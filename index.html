My Chat uses a browser plugin called FrankerFaceZ (or for short FFZ). It’s an alternative to BetterTwitchTV (or BTTV) that enhances your Twitch experience, while it also allows for custom emotes for non-affiliated and/or partnered streams. A link for a download can be found here.
